Bron-Kerbosch Algorithm

This algorithm is a variant of the Bron–Kerbosch algorithm optimized for sparse graphs by utilizing the concept of degeneracy. Degeneracy is a measure of how sparse a graph is, defined as the smallest number d such that every subgraph contains a vertex of degree at most d. The algorithm lists all maximal cliques in time O((d⋅n⋅3)^d/3), which is near-optimal for sparse graphs.

Tomita Algorithm

The Tomita algorithm is a depth-first search algorithm used to generate all maximal cliques of an undirected graph. It incorporates pruning methods similar to the Bron–Kerbosch algorithm to improve efficiency. The algorithm outputs the maximal cliques in a tree-like form, which optimizes space usage. Its worst-case time complexity is O(3^(n/3)) for an n-vertex graph, making it optimal as there can be at most O(3^(n/3)) maximal cliques in the graph. Extensive computational experiments demonstrate its superior performance compared to previous algorithms.
